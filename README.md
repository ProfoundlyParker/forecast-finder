## Description
<p>A Next.js app that displays weather data based on the selected location. Built with <strong>Next.js</strong>, <strong>React</strong>, <strong>TypeScript</strong>, <strong>React hooks</strong>, <strong>Tailwind CSS</strong>, <strong>Axios</strong>, <strong>Jotai</strong>, <strong>Date-fns</strong>, <strong>Clsx</strong>, <strong>Tanstack/react-query</strong>, <strong>OpenWeatherMap API</strong>, and deployed on <strong>Vercel</strong></p><br>
